How PGP Works

Major parts of PGP

  • Keys
    • Users create their own Public/Private keypairs
    • Users can have multiple keys
    • Users maintain a keyring of cached public keys
  • Messages
    • Encrypt messages to people "using their Public Key"
    • Sign messages using your Private Key
    • Decrypt/Verify messages
  • Key Management
    • Add/remove keys from the keyring
    • Sign a key
    • Extract public keys to send to others (exchange keys)
    • Fingerprint keys (to help verify keys)

Keyservers:

  • A way to exchange PGP Public Keys
  • Email Interface: Send mail to pgp-public-keys@pgp.mit.edu
  • For more information, there read the Keyserver Help File
